# Ich-polityka.
Polityka zegarów i ekologów to afery dzieci z duanem i sychizofremią dzieci indygo i dzieci kiwaty żyją w polityce kłamstw i niedomówień politycznych ustaw i maxymalnym braku luksusu i nie konformtowym minimalizmie religi judaistycznych kłamców kabały i maxymalnej drodze do minimalizmu i braku pieniedzy i gołej i nagiej miłości.
